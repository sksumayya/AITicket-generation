AI Ticket API Documentation

Integrate and manage tickets using our REST API

Base URL

http://127.0.0.1:5000/api/
GET
/api/tickets

Retrieve a list of all tickets.

Response Example:
[ { "ticket_id": 1, "title": "Login Issue", "category": "Authentication", "priority": "High", "status": "Open", "created_at": "2026-02-20" } ]
GET
/api/tickets/{id}

Retrieve a specific ticket by ID.

Response Example:
{ "ticket_id": 1, "title": "Login Issue", "description": "User cannot login", "category": "Authentication", "priority": "High", "status": "Open" }
POST
/api/tickets

Create a new ticket.

Request Body:
{ "title": "Server Error", "description": "Application crashes", "category": "Technical", "priority": "High" }
Response:
{ "message": "Ticket created successfully" }
DELETE
/api/tickets/{id}

Delete a ticket by ID.

Response Example:
{ "message": "Ticket deleted successfully" }

HTTP Status Codes

  • 200 – Success
  • 201 – Created
  • 400 – Bad Request
  • 404 – Not Found
  • 500 – Internal Server Error
