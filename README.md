AI Powered IT Ticket Creation and Categorization

Project Overview
This project is an AI based IT helpdesk ticket system that automatically creates and categorizes support tickets using transformer based natural language processing. The application analyzes customer queries and predicts the appropriate category to help support teams manage and resolve issues more efficiently. The system is built using Python and Flask with a simple web interface.

Features
Automatic ticket creation and categorization
Transformer based NLP model
Flask based web application
Simple frontend using HTML and CSS
Support for multilingual customer queries

Tech Stack
Backend: Python, Flask
AI and NLP: Transformers, Machine Learning
Frontend: HTML, CSS, javascript
Development Tools: VS Code, Kaggle

Dataset
This project uses the Multilingual Customer Support Tickets dataset from Kaggle.
Link: https://www.kaggle.com/datasets/tobiasbueck/multilingual-customer-support-tickets

The dataset contains multilingual customer support conversations and ticket information that are used to train the NLP model for automatic ticket categorization.
